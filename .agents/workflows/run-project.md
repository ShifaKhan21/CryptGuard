---
description: How to start the CryptGuard project
---
1. Start the Backend API Server:
   - Navigate to `Packet_analyzer`.
// turbo
   - Run: `python api_server.py`

2. Start the Frontend Web UI:
   - Navigate to `Packet_analyzer/web_ui`.
// turbo
   - Run: `npm run dev`

3. Access the application at `http://localhost:5173`.
